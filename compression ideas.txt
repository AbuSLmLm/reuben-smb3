Compression Commands:
XY  PP ZZ ZZ...

if X < Y
    X = start writing offset
    Y = end writing offset
    ZZ = data to write

if X == F && Y < E
    Skip Y rows

if X == E && Y == 0
    Fill row with ZZ

if X == E && Y > 0
    Starting at Y, fill rows with ZZ

Level Header
CCCC CCCC = Clear level with value;
GGGG GGGG = Graphics bank;
PPPP PPPP = Palette Index;
SSSS TTTT = StartAction, Length;
XXXX XXXX = X Position;
YYYY YYYY = Y Starting position;
MMMM MMMM = M - music, L = length (up to 0xE);
TTTT TTTT = Time (times 10, up to 999);
VYSS LLLL = V-Scroll, Y Extended Y position, Scroll Type, Level type;
UUUU UUUU = Unused 1;
UUUU UUUU = Unused 2;
UUUU UUUU = Unused 3;


Pointer data...
IIII IIII
XXXX XXXX = X Enter
YYYY YYYY = Y Enter
XXXX XXXX = X Exit
YYYY YYYY = Y Exit
UUUU UTTT = Exit Type


Map Header
GGGG GGGG - Graphics
PPPP PPPP - Palette index
XXXX YYYY - X-start, Y-Start
MMMM MMMM - Music
PPPP SSSS - Page, Scroll
UUUU UUUU - Unused

Map pointer data
IIII IIII = Level index
XXXX XXXX = X Enter
YYYY YYYY = Y Enter

$A0xx-$DFxx
%CCPP PPPP = CC = command number, PP PPPP = paramter
DD

CC = 00 - skip
CC = 01 - skip row
CC = 10 - write
CC = 11 - repeat data

maximum data size: 6620
maximum sprite size: 769


NOTEs
RAM
4E5 - Level music storage
545 - Starting Action
70A - Current Level Type
727 - Current World
75 - Current Scroll position (also with map)
79 - Current x position (map)
90 - current x position
22 - Level length
375 - scroll type
567 - P-Switch Time

ROM
0x11653 - P-Switch Time
0x14F44 - Table for map lengths in the format LLLL xxxx where LLLL is the length and xxxx is unused


Reuben ROM Map:

0x15610-0x1501F - Map lengths table

0x3C010-0x3E00F - Palettes
0x3E010-0x3FC0F - TSA data
0x18C10-0x1800F - Level index table
0x40010-0x4400F - Map/Sprite data
0x44010-0xFD00F - Level/Sprite data
0xFD010-0xFE00F - Project use
